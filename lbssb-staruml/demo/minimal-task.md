# Minimal Task

Use `lbssb-staruml` to produce one class diagram from a source `.mdj`.

Required delivery:

- Editable StarUML working `.mdj`.
- One PNG exported from StarUML.
- `diagram-manifest.json`.
- `.lbssb/preflight-report.json`.
- `.lbssb/verification-report.json`.

Verified is allowed only when capability level is `L4` and `verify_deliverables.py` exits `0`.
